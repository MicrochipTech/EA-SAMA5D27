# SAMA5D27 TFBGA289
## Early Access Repository

The SAMA5D27 is a high-performance, low-power Arm Cortex-A5 CPU-based embedded microprocessor (MPU) running up to 500 MHz, with support for multiple memories such as DDR2, DDR3L, LPDDR2, LPDDR3, and QSPI and e.MMC Flash. The device integrates powerful peripherals for connectivity and user interface applications, and offers advanced security functions (Arm TrustZoneÂ® , tamper detection, secure data storage, secure boot, etc.), as well as high-performance crypto accelerators AES, SHA and TRNG.

The SAMA5D27(T)-V/8GW MPU is offered in a compact 9x9x1.2 mm TFBGA289 package with a 0.5mm pitch and is now available to Early Access customers.
